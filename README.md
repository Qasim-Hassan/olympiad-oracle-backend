# Oracle-backend

This serves as the backend code for: https://olympiadhub-pk.netlify.app
Written in python, it serves the AI-model for the olympiad oracle on the site.

----

## Setup on your machine

Clone the repository, run:

> pip install Flask

to install flask. Then run the app with:

> python app.py

and open *localhost:5000* to see the app running.
